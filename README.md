# Vidya Music (Compose rewrite)

Vidya Music is a player for the [Vidya Intarweb Playlist](https://www.vipvgm.net/) (aka VIP Aersia).

This version is a complete rewrite in Jetpack Compose of the original Flutter version,
one of the main additions is Material 3 support along with many QoL changes.

## AI Usage Disclaimer

Development of this application makes heavy usage of AI generation tools (more specifically Google Gemini). 

GenAI is used to, among other reasons:

* Help make sure the project adheres to Clean Arch while keeping the intent of the app
* Help making sure the app adheres to Material 3 guidelines 
* Help managing the interfacing between Media3 (the playback backend) and the app 
* Help debugging random issues that may come up from Android or Compose
(at least from the point of view of a Flutter Developer)
